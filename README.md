# scanplan

Implementation of the paper "Automating adaptive scan planning for static laser scanning in complex 3D environments" [1]. This project provides tools for processing and analyzing 3D scene data and provide an optimized solution for the given environment.

> **Note**: This is a work in progress. The codebase is currently being cleaned up and documented, first release coming 07/2025.

## Citation

If you use this code in your research, please cite:

```
[1] Noichl, F., Lichti, D.D., and Borrmann, A. (2024). Automating adaptive scan planning for static laser scanning in complex 3D environments. 
Automation in Construction, https://doi.org/10.1016/j.autcon.2024.105511
```
